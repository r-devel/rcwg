- Hugo has been working on blog post about this to describe issue and evaluate impact of using braces in function definition that just calls standardGeneric, e.g. performance, debugging

### Compression

- zstdfile(), xzfile() allow to control level of compression, but not possible if compressing data in memory with memCompress().

    - Some discussion on https://github.com/r-devel/r-dev-day/issues/89, but better to open issue on Bugzilla to take forward as requires more discussion by R Core.
